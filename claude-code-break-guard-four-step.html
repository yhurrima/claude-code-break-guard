How it works

A four-step process.

01

Install Hook

安装脚本写入 ~/.claude/settings.json，把命令挂到 UserPromptSubmit。每次新 prompt 提交前，先进入本地守门逻辑。

02

Intercept Prompt

hook 读取 stdin 里的 prompt，同时读取 config.json 和 state.json。如果工作时间没超，就允许继续。

03

Verify Idle

通过 ioreg 读取 macOS 的 HIDIdleTime。键鼠触控板有输入，空闲片段就会被打断，不算真实休息。

04

Allow / Block

休息达标返回 exit 0 放行；休息不够返回 exit 2 阻断，并提示还需要多少真实空闲时间。
